    COMANDOS TERMINAL (sin el guion xd):
    -git init: comando para iniciar el proyecto una vez
    -git config --global user.name Salinas-Luis: para poder configurar en la maquina actual ese usuario (en este caso Salinas-Luis)
    -git config --global user.email "lsalinasr2300@alumno.ipn.mx"
    -git add . : agregar todo el archivo o todos los archivos ( "." para todo y nombre de archivo nomas para el archivo)
    -git status: para saber que estamos haciendo
    -git remote add origin https://github.com/Salinas-Luis/5IV8_Salinas_RamosLuis_PWP.git: Sirve para conectar el repositorio de la nube con el local
    -git branch nombre: Crear rama (-M sirve para crearla y meterse: por ejemplo git branch -M main (crea y metete a main) )
    -git push origin main: subir el cambio, (en este caso es main pq es el nombre de la rama)
    -git commit: es la referencia del cambio, SE NECESITA USAR DESPUES DE GIT ADD Y ANTES DEL GIT PUSH
    -git commit -m "Poner lo que hicimos (actualize el registro, cambie la version, etc)": ejemplo de uso, adentro de las comillas va lo que hiciste
